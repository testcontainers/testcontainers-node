# Mockserver Module

[MockServer](https://www.mock-server.com/#what-is-mockserver) allows you to mock any server or service via HTTP or HTTPS, such as a REST or RPC service.

## Install

```bash
npm install @testcontainers/mockserver --save-dev
```
